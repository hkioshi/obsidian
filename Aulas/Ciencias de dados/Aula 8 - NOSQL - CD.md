# Bancos relacionais
- ex:
	 - mysql
	 - oracle
	 - sqlserver
	 - postgress
- Caracteristicas.
	- estrutura rígida
	- ruim pra armazenar, indexar volume grandes
	- Dificuldades para escalar verticalmente
	- relacionamentos grandes são problemáticos
